# @uppercase-letters/j

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **J**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/j
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseJ = require('@uppercase-letters/j')

console.log(uppercaseJ) // Output: "J"
```
