# @uppercase-letters/k

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **K**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/k
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseK = require('@uppercase-letters/k')

console.log(uppercaseK) // Output: "K"
```
