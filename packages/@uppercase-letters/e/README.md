# @uppercase-letters/e

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **E**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/e
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseE = require('@uppercase-letters/e')

console.log(uppercaseE) // Output: "E"
```
