# @uppercase-letters/q

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **Q**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/q
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseQ = require('@uppercase-letters/q')

console.log(uppercaseQ) // Output: "Q"
```
