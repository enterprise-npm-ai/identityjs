# @uppercase-letters/b

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **B**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/b
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseB = require('@uppercase-letters/b')

console.log(uppercaseB) // Output: "B"
```
