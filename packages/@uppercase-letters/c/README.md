# @uppercase-letters/c

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **C**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/c
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseC = require('@uppercase-letters/c')

console.log(uppercaseC) // Output: "C"
```
