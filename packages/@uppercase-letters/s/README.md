# @uppercase-letters/s

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **S**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/s
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseS = require('@uppercase-letters/s')

console.log(uppercaseS) // Output: "S"
```
