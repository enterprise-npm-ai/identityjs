# @uppercase-letters/x

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **X**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/x
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseX = require('@uppercase-letters/x')

console.log(uppercaseX) // Output: "X"
```
