# @uppercase-letters/a

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **A**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/a
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseA = require('@uppercase-letters/a')

console.log(uppercaseA) // Output: "A"
```
