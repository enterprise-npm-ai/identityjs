# @uppercase-letters/t

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **T**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/t
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseT = require('@uppercase-letters/t')

console.log(uppercaseT) // Output: "T"
```
