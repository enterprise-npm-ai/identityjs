# @uppercase-letters/g

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **G**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/g
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseG = require('@uppercase-letters/g')

console.log(uppercaseG) // Output: "G"
```
