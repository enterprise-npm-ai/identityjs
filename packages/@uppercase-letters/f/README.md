# @uppercase-letters/f

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **F**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/f
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseF = require('@uppercase-letters/f')

console.log(uppercaseF) // Output: "F"
```
