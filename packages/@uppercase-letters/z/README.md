# @uppercase-letters/z

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **Z**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/z
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseZ = require('@uppercase-letters/z')

console.log(uppercaseZ) // Output: "Z"
```
