# @uppercase-letters/w

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **W**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/w
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseW = require('@uppercase-letters/w')

console.log(uppercaseW) // Output: "W"
```
