# @uppercase-letters/v

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **V**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/v
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseV = require('@uppercase-letters/v')

console.log(uppercaseV) // Output: "V"
```
