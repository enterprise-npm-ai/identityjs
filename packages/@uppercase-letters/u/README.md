# @uppercase-letters/u

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **U**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/u
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseU = require('@uppercase-letters/u')

console.log(uppercaseU) // Output: "U"
```
