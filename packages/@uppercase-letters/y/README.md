# @uppercase-letters/y

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **Y**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/y
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseY = require('@uppercase-letters/y')

console.log(uppercaseY) // Output: "Y"
```
