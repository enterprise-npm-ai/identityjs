# @uppercase-letters/h

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **H**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/h
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseH = require('@uppercase-letters/h')

console.log(uppercaseH) // Output: "H"
```
