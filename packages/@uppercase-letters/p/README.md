# @uppercase-letters/p

## Overview

This is a lightweight, single-purpose utility package that exports the uppercase letter **P**. 

It is part of the uppercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @uppercase-letters/p
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const uppercaseP = require('@uppercase-letters/p')

console.log(uppercaseP) // Output: "P"
```
