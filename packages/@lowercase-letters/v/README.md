# @lowercase-letters/v

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **v**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/v
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseV = require('@lowercase-letters/v')

console.log(lowercaseV) // Output: "v"
```
