# @lowercase-letters/q

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **q**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/q
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseQ = require('@lowercase-letters/q')

console.log(lowercaseQ) // Output: "q"
```
