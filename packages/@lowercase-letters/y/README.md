# @lowercase-letters/y

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **y**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/y
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseY = require('@lowercase-letters/y')

console.log(lowercaseY) // Output: "y"
```
