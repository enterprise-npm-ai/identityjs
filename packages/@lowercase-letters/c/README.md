# @lowercase-letters/c

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **c**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/c
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseC = require('@lowercase-letters/c')

console.log(lowercaseC) // Output: "c"
```
