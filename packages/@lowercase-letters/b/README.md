# @lowercase-letters/b

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **b**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/b
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseB = require('@lowercase-letters/b')

console.log(lowercaseB) // Output: "b"
```
