# @lowercase-letters/p

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **p**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/p
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseP = require('@lowercase-letters/p')

console.log(lowercaseP) // Output: "p"
```
