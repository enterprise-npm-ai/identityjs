# @lowercase-letters/e

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **e**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/e
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseE = require('@lowercase-letters/e')

console.log(lowercaseE) // Output: "e"
```
