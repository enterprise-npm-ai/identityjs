# @lowercase-letters/t

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **t**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/t
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseT = require('@lowercase-letters/t')

console.log(lowercaseT) // Output: "t"
```
