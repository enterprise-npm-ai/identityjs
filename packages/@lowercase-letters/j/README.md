# @lowercase-letters/j

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **j**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/j
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseJ = require('@lowercase-letters/j')

console.log(lowercaseJ) // Output: "j"
```
