# @lowercase-letters/k

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **k**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/k
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseK = require('@lowercase-letters/k')

console.log(lowercaseK) // Output: "k"
```
