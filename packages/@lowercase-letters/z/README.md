# @lowercase-letters/z

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **z**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/z
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseZ = require('@lowercase-letters/z')

console.log(lowercaseZ) // Output: "z"
```
