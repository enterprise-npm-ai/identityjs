# @lowercase-letters/x

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **x**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/x
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseX = require('@lowercase-letters/x')

console.log(lowercaseX) // Output: "x"
```
