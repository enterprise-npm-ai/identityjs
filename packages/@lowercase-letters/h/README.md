# @lowercase-letters/h

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **h**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/h
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseH = require('@lowercase-letters/h')

console.log(lowercaseH) // Output: "h"
```
