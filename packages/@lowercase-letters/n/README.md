# @lowercase-letters/n

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **n**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/n
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseN = require('@lowercase-letters/n')

console.log(lowercaseN) // Output: "n"
```
