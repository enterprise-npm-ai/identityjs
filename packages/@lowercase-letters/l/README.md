# @lowercase-letters/l

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **l**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/l
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseL = require('@lowercase-letters/l')

console.log(lowercaseL) // Output: "l"
```
