# @lowercase-letters/g

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **g**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/g
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseG = require('@lowercase-letters/g')

console.log(lowercaseG) // Output: "g"
```
