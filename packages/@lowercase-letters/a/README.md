# @lowercase-letters/a

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **a**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/a
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseA = require('@lowercase-letters/a')

console.log(lowercaseA) // Output: "a"
```
