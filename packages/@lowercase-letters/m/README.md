# @lowercase-letters/m

## Overview

This is a lightweight, single-purpose utility package that exports the lowercase letter **m**. 

It is part of the lowercase-letters project aiming to provide modular access to the English alphabet.

## Installation

```bash
npm install @lowercase-letters/m
```

## Usage

Import this package to use the letter in your application without hardcoding strings.

```javascript
const lowercaseM = require('@lowercase-letters/m')

console.log(lowercaseM) // Output: "m"
```
